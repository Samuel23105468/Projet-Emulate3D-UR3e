## Introduction
This project aims to emulate the functionalities of a UR3e robotic arm using Emulate3D.

## Installation
```bash
git clone https://github.com/Samuel23105468/Projet-Emulate3D-UR3e.git
cd Projet-Emulate3D-UR3e
```

## Usage
Run the program with the following command:
```bash
python main.py
```
